# Лабораторна робота №3 — REST API веб-застосунок

## Запуск проекту

1. Створити та активувати віртуальне середовище
# Windows PowerShell
python -m venv venv
.\venv\Scripts\Activate.ps1

# Linux / macOS
python -m venv venv
source venv/bin/activate
2. Встановити Flask:
pip install flask
3. Запуск проекта:
python app.py

- Сервер запуститься на http://127.0.0.1:5000
- Веб-інтерфейс: /notes
- REST API: /api/notes
